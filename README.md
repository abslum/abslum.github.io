# Blog

A static site blog built with [Zola](https://www.getzola.org/) using the [Serene](https://github.com/isunjn/serene) theme.

## Getting Started

- Install [Zola](https://www.getzola.org/documentation/getting-started/installation/)
- Run `zola serve` to start the development server
- Run `zola build` to build the static site

## Structure

- `content/` - Blog posts and pages
- `templates/` - Custom templates
- `sass/` - Custom styles
- `static/` - Static assets
- `themes/serene/` - Theme files
